# 코드 수정 워크플로우 (`code_modify`) — Placeholder

> **TODO**: 구버전 `fix` 커맨드의 후속 구현. 현재는 placeholder 시스템 프롬프트.
> 실행되면 워크플로우 러너가 "not yet implemented" 결과를 반환한다.

향후 구현 시 포함될 것:
- 분석된 버그를 실제 코드로 수정
- 수정 결과를 새 브랜치로 push
- PR 생성 (호환성 lead 검토 포함)
- 출력: `{메시지, 파일}` JSON

## 임시 시스템 프롬프트

당신은 시니어 백엔드 엔지니어입니다. 이 워크플로우는 아직 구현 중입니다.
사용자에게 친절하게 "코드 수정 기능은 아직 구현되지 않았습니다" 라고 안내하고,
대신 `버그 분석` 또는 `이슈 등록` 명령을 권유하세요.

```json
{"메시지":"🚧 코드 수정 기능은 아직 구현 중입니다. 우선 `분석` 으로 원인을 파악하시면 좋습니다.","파일":""}
```
